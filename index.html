How to Measure (Accurate Method)

Take a t-shirt you already wear comfortably.
Lay it flat → measure from armpit to armpit → multiply by 2.
Match that number with the chest size above.

Fit Guidance (Real-world)

• If you wear M in brands like Zudio / H&M → choose M

• If you like slightly loose → go one size up

• If you want oversized streetwear look → go 1 size up

• If unsure between two sizes → ALWAYS go bigger

Important

This is an oversized fit. Do NOT size down unless you want a tight fit.
